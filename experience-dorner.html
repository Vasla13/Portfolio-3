Dorner - Update Verwaltung

Application web de gestion des mises a jour logicielles permettant l'administration de versions, l'automatisation de l'installation et la simplification du deploiement via SQLite (sans configuration MySQL separee).

PHP SQLite Bash Apache Admin Tools
Entreprise

DORNER Health IT

Periode

2025 - Stage (8 semaines)

Role

Developpeur Web Securise

Fonctionnalites

Authentification utilisateurs et administrateurs

  • Inscription, connexion et deconnexion.
  • Gestion des droits selon le profil utilisateur.

Gestion des versions

  • Affichage, modification et suppression des versions logicielles.
  • Televersement d'archives (en local ou via URL).

Processus d'installation (3 etapes)

  1. Extraction de l'archive.
  2. Configuration (execution d'un script d'installation).
  3. Deplacement des repertoires pertinents (imed-Info et imed-Admin) dans le dossier de version.

Fonction de lien court

  • Generation d'un token court pour rediriger directement vers la version installee.

Gestion simplifiee de la base de donnees

  • Utilisation de SQLite via PDO.
  • Aucune configuration de base de donnees separee n'est necessaire.

Prerequis

  • Serveur web: Apache, Nginx ou autre serveur compatible PHP.
  • PHP: version 7.0 ou superieure (7.4+ recommandee).
  • SQLite: integre a PHP, aucune installation supplementaire.
  • Shell: Bash (pour l'execution du script d'installation).
  • Droits de fichiers: ecriture requise sur uploads/ et versions/.

Ce projet m'a permis de travailler sur une application deploiement/update orientee production, avec une attention forte sur la fiabilite du processus d'installation, la securite des acces et la simplicite d'exploitation pour les administrateurs.
